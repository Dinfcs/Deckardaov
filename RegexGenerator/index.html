How to Use This Tool

  • Enter one or more examples of the text you want to match in the box below. Place each example on a new line.
  • Crucial Rule: For this simple generator to work, all your examples must have the exact same length.
  • The tool analyzes the character type (letter, number, symbol) at each position across all examples to infer the common pattern.
  • Example: If you enter item-001 and item-002, the tool will generate item-\d{3}.
